TerraComposito is a global magic mod for Minecraft on Forge (for now). It takes inspiration from various awesome mods like Botania, Nature’s Aura, and others.

I want to create a mod that represents the pinnacle of magical technology for different modpacks. For example, it might include an armor set that functionally rivals MekaSuit but is not the peak of technology, but rather the peak of magic.

I’m developing this mod in my free time as a hobby, but I’d be thrilled if someone invited me to work on a cool project.
